## 🚀 Getting Started

### Prerequisites
- Basic understanding of data structures (arrays, linked lists, trees, graphs)
- Familiarity with a programming language (Python, Java, C++, JavaScript, etc.)
- 30-45 minutes daily for problem-solving

### Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SwatiNH/dsa-daily-solver.git
   cd dsa-daily-solver
   ```

2. **Explore the DSA templates:**
   - All problem links are available in `dsa-templates.json`
   - Problems are sourced from LeetCode and similar platforms

3. **Daily Practice Workflow:**
   - Pick a category (or use random selection)
   - Solve the problem without looking at solutions
   - Implement your approach in your preferred language
   - Create a folder/branch with your solution
   - Submit a PR for community feedback (optional)

## 📖 How to Use with GitHub Copilot

The repository is structured to work seamlessly with GitHub Copilot skills. Ask Copilot:

- **"Give me a random DSA problem from today"** → Copilot will select a random problem across all categories
- **"Show me a sliding window problem"** → Copilot will suggest a problem from that category
- **"What's my next problem to solve?"** → Copilot provides problem rotation

## 🎓 Difficulty Levels

- **Easy**: Fundamentals and basic applications (1-2 weeks in)
- **Medium**: Core concepts and real-world scenarios (weeks 2-4)
- **Hard**: Advanced optimization and complex patterns (weeks 4+)

## 💡 Study Tips

1. **Start Easy**: Begin with Array and Sliding Window categories
2. **Pattern Recognition**: Look for similar patterns across categories
3. **Time Complexity**: Always analyze and optimize for time/space complexity
4. **Teach Others**: Explain your solution to reinforce learning
5. **Consistency**: Solve one problem daily for 30 days minimum
6. **Track Progress**: Maintain a log of problems solved
